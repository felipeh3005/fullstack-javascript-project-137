## RSS Feed Aggregator

RSS Feed Aggregator is a production-oriented JavaScript web application that allows users to add, validate, and monitor RSS feeds through a clean and interactive interface.  
The project includes asynchronous feed updates, modal-based post previews, read-state tracking, automated testing, and CI-driven quality control.
